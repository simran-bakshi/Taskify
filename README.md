## 📌 About the Project

This is a **task management web application 📝** built using Django 🌿 and DRF that allows users to:

* 🔐 Create an account & securely log in  
* 🎟️ Authenticate using JWT tokens  
* ✅ Create and manage tasks  
* ✏️ Update task details  
* ❌ Delete tasks  
* 📋 View tasks specific to each user  

The focus of this project is not just functionality, but also understanding:

* 🧱 Models  
* ⚡ Django REST Framework  
* 🔐 Authentication & Authorization  
* 🔗 URL routing  
* 🛡️ Secure user-based data handling

## 📂 Project Setup

    # Clone the repository
    git clone https://github.com/simran-bakshi/Taskify

    # Navigate into the project folder
    cd todoproject

    # Create virtual environment
    python -m venv venv

    # Activate virtual environment (Windows)
    .venv\Scripts\activate

    # Install dependencies
    pip install -r requirements.txt

    # Run migrations
    python manage.py migrate

    # Start development server
    python manage.py runserver
