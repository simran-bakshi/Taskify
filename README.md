This project marks my first step into the world of **Django 🌐**, coming from a background in Java ☕ development.

## 📌 About the Project

This is a basic **Todo App 📝** built using Django 🌐 that allows users to:

* ✅ Add tasks
* ✏️ Update tasks
* ❌ Delete tasks
* 📋 View all tasks

The focus of this project is not just functionality, but also understanding Django concepts like:

* 🧱 Models
* 🎯 Views
* 🧾 Templates
* 🔗 URL routing
* ⚡ Generic views

## 📂 Project Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone <your-repo-link>

# Navigate into the project folder
cd <project-folder>

# Create virtual environment
python -m venv venv

# Activate virtual environment
# Windows
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run migrations
python manage.py migrate

# Start development server
python manage.py runserver
```
